User diwajibkan login untuk bisa mengikuti tryout online, jika user belum memeliki akun, user dapat register terlebih dahulu

Setiap quiz memiliki kategori dan waktunya masing-masing, jika user masih berada di kategori Bahasa Indonesia yang bukan kategori terakhir lalu waktunya sudah habis, maka user akan dipaksa untuk lanjut ke kategori selanjutnya. User dapat melakukan previos soal selama masih di dalam kategori yang sama namun tidak bisa ke kategori sebelumnya.

Ini adalah hasil ketika user selesai mengerjakan tryout, semua hasil dapat dilihat dan didownload dalam bentuk PDF
